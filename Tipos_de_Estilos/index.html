Estilo externo

Este tipo de estilo es el que se realiza por medio de un archivo de tipo CSS3, que se enlaza a un archivo html o php por medio del código "link".

Estilo interno

Por parte del estilo interno, se localiza en la parte del archivo html en la etiqueta principal de "head", donde tenemos que abrir la etiqueta "style" y escribir como si se tratase de una hoja de estilo CSS3.

Estilo inline

Para concluir con el estilo inline, es uno que se realiza justamente en el momento en el que se escribe una etiqueta en el archivo html, pero se dease editarla específicamente a esta en la misma línea que se haya decidido de la siguiente forma.
